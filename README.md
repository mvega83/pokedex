## Descripción
1.- Clonar el repositorio
2.- Ejecutar
```
Yarn install
```
3.- Tener NESt CLI instalado
```
npm i-g @nestjs/cli
```

4.- Levantar la base de datos

```
docker-composer up -d
```

### Stack
* Mongo DB
* Nest


[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
